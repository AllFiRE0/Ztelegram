# ZTelegram

![ZTelegram Logo](https://i.ibb.co/tpGFZBxj/1.jpg)

**ZTelegram** - мощный и профессиональный плагин для интеграции Minecraft сервера с Telegram. Плагин создает надежный мост между вашим сервером и Telegram, предоставляя двустороннюю синхронизацию чата, систему регистрации игроков, статистику, репутацию, переводы денег, мини-игры и многое другое.

---

## ⚠️ ВАЖНО: Обязательная зависимость!

**🔴 ПЛАГИН НЕ БУДЕТ РАБОТАТЬ БЕЗ VAULT!**

Плагин **ZTelegram** требует установки плагина **Vault** для работы с экономикой сервера. Без Vault следующие функции не будут работать:
- Команда `/topbal` (топ по балансу)
- Система переводов денег `/pay`
- Награды за регистрацию (если используются команды экономики)
- Награды в рулетке (если используются команды экономики)
- Автоматические награды за топы

**Установка Vault:**
1. Скачайте Vault с [SpigotMC](https://www.spigotmc.org/resources/vault.34315/)
2. Поместите файл `Vault.jar` в папку `plugins`
3. Перезапустите сервер
4. Установите плагин экономики (например, EssentialsX, CMI, или другой)

## 🌟 Основные функции

- **📱 Двусторонняя синхронизация чата** - сообщения из Minecraft отображаются в Telegram и наоборот
- **🔔 Уведомления о событиях** - вход/выход игроков, смерти, запуск/остановка сервера
- **👮 Консольный доступ** - выполнение команд сервера через Telegram (только для администраторов)
- **👤 Система регистрации** - привязка аккаунтов Minecraft к Telegram с наградами
- **⭐ Система репутации** - игроки могут оценивать друг друга положительно или отрицательно
- **💸 Система переводов** - перевод денег между игроками через Telegram
- **🎲 Рулетка** - случайные награды для игроков онлайн
- **📊 Статистика и топы** - детальная статистика игроков, топы по времени игры и балансу
- **🎮 Мини-игра "Угадай слово"** - интерактивная игра с наградами
- **📱 Интерактивное меню** - удобный интерфейс для всех функций плагина
- **🛡️ Управление доступом** - белые и черные списки для контроля доступа
- **⏰ Автоматические уведомления** - ежедневные топы и сводки
- **📅 Планировщик задач** - автоматическое выполнение команд по расписанию
- **🔄 Система рестарта** - перезагрузка сервера с таймером и предупреждениями

## 📋 Требования

### Обязательные требования:

- **Java 21** или выше
- **Minecraft Server** (Paper/Spigot) версии **1.17+** (полная поддержка до **1.21.11**)
- **Vault** - **ОБЯЗАТЕЛЬНО!** Плагин не будет работать без Vault
- **Telegram бот** с токеном от [@BotFather](https://t.me/BotFather)
- **Разрешенное взаимодействие** с API Telegram (без прокси в большинстве регионов)
- **Минимум 512MB RAM** для работы плагина

### Рекомендуемые плагины:

- **Плагин экономики** (EssentialsX, CMI, или другой) для работы с балансом
- **Плагин прав** (LuckPerms, PermissionsEx) для управления правами доступа

## 💻 Установка

### Шаг 1: Установка Vault

1. Скачайте **Vault** с [SpigotMC](https://www.spigotmc.org/resources/vault.34315/)
2. Поместите файл `Vault.jar` в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Убедитесь, что Vault загрузился успешно (проверьте консоль)

### Шаг 2: Установка ZTelegram

1. Скачайте последнюю версию плагина из [Releases](https://github.com/Z-oobastik-s/Ztelegram/releases)
2. Поместите файл `ZTelegram.jar` в папку `plugins` вашего сервера
3. Перезапустите сервер
4. При первом запуске будут созданы конфигурационные файлы:
   - `plugins/Ztelegram/config.yml` - основные настройки
   - `plugins/Ztelegram/game.yml` - настройки мини-игры
   - `plugins/Ztelegram/ztelegram.db` - база данных SQLite (если включена)

### Шаг 3: Создание Telegram бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Укажите имя бота (например: "My Server Bot")
   - Укажите username бота (должен заканчиваться на `bot`, например: `myserver_bot`)
4. После создания вы получите **токен бота** (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. **Сохраните токен** - он понадобится для настройки

### Шаг 4: Создание каналов Telegram

Создайте каналы/группы в Telegram для разных функций:

1. **Основной канал** - для чата между игрой и Telegram
   - Создайте канал или группу
   - Добавьте бота как администратора
   - Скопируйте ID канала (см. ниже)

2. **Консольный канал** - для выполнения команд (опционально)
   - Создайте группу
   - Добавьте бота как администратора
   - Скопируйте ID группы

3. **Канал регистрации** - для регистрации игроков
   - Создайте канал или группу
   - Добавьте бота как администратора
   - Скопируйте ID канала

4. **Игровой канал** - для мини-игры (опционально)
   - Создайте канал или группу
   - Добавьте бота как администратора
   - Скопируйте ID канала

5. **Канал статистики** - для автоматических уведомлений (опционально)
   - Создайте канал или группу
   - Добавьте бота как администратора
   - Скопируйте ID канала

### Шаг 5: Получение ID каналов

**Способ 1: Через бота**
1. Добавьте бота в канал/группу
2. Отправьте любое сообщение в канал/группу
3. Откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
4. Найдите в ответе `"chat":{"id":-1001234567890}` - это и есть ID канала
5. Для **тем (topics)** в каналах ID будет выглядеть как `-1001234567890_123456`

**Способ 2: Через бота @userinfobot**
1. Добавьте бота @userinfobot в канал/группу
2. Отправьте команду `/start`
3. Бот покажет ID канала

**Способ 3: Через веб-версию Telegram**
1. Откройте канал/группу в веб-версии Telegram
2. Посмотрите URL - ID будет в адресе

## ⚙️ Быстрая настройка

После установки откройте файл `plugins/Ztelegram/config.yml` и настройте минимально необходимые параметры:

```yaml
# Токен бота от BotFather
bot:
  token: 'ВАШ_ТОКЕН_ЗДЕСЬ'

# ID каналов
channels:
  main: '-1001234567890'        # Основной канал
  console: '-1001234567890'     # Консольный канал (можно оставить пустым)
  register: '-1001234567890'    # Канал регистрации
  game: ''                      # Игровой канал (опционально)
  statistics: ''               # Канал статистики (опционально)

# ID администраторов (ваш Telegram ID)
administrators:
  telegram_ids:
    - 123456789                 # Ваш Telegram ID
```

После настройки перезапустите сервер или выполните команду `/telegram reload` в игре.

#### Команда репутации `/rep`

| Команда | Описание | Право доступа | Пример |
|---------|----------|---------------|--------|
| `/rep <player>` | Показывает репутацию игрока | `ztelegram.reputation.use` | `/rep PlayerName` |
| `/rep +<player> [причина]` | Дает положительную репутацию игроку | `ztelegram.reputation.use` | `/rep +PlayerName За помощь новичкам` |
| `/rep -<player> [причина]` | Дает отрицательную репутацию игроку | `ztelegram.reputation.use` | `/rep -PlayerName Нарушение правил` |
| `/rep top [тип]` | Показывает топ игроков по репутации | `ztelegram.reputation.use` | `/rep top positive` |
| `/rep stats` | Показывает статистику системы репутации | `ztelegram.reputation.use` | `/rep stats` |
| `/rep reset <player>` | Сбрасывает репутацию игрока (только админы) | `ztelegram.reputation.admin` | `/rep reset PlayerName` |

**Типы топа репутации:**
- `total` - по общему рейтингу (по умолчанию)
- `positive` - по положительной репутации
- `negative` - по отрицательной репутации
- `percentage` - по проценту положительной репутации

## 🔧 Системы плагина

### Система регистрации

Система регистрации позволяет привязать аккаунт Minecraft к Telegram аккаунту. После регистрации игрок может:
- Отправлять сообщения в игру через Telegram
- Получать уведомления о событиях
- Использовать все функции плагина
- Получить награду за регистрацию

#### Способы регистрации:

**1. Регистрация через никнейм:**
1. Игрок заходит на сервер под своим никнеймом
2. В Telegram отправляет свой никнейм в канал регистрации (например: `PlayerName`)
3. Если игрок онлайн, регистрация происходит автоматически
4. Игрок получает награду и уведомление в игре

**2. Регистрация по коду:**
1. Игрок в игре выполняет команду `/telegram link`
2. Получает 6-8 значный код (действует 10 минут по умолчанию)
3. Отправляет этот код в канал регистрации
4. Регистрация происходит автоматически
5. Игрок получает награду и уведомление

**3. Регистрация через меню:**
1. Игрок отправляет `/menu` в канале регистрации
2. Выбирает "Регистрация" в меню
3. Следует инструкциям

#### Настройка наград за регистрацию:

В `config.yml` в секции `register-channel.reward-commands` можно настроить команды, которые выполнятся при регистрации:

```yaml

    - "eco give %player% 500"                    # Дать 500 монет
    - "xp give %player% 20l"                     # Дать 20 уровней опыта
    - "give %player% iron_sword 1"               # Дать железный меч
    - "broadcast &b%player% &eзарегистрировался" # Сообщение в чат
```

**Плейсхолдеры:**
- `%player%` - никнейм игрока


## 🔧 Системы плагина

### Система чек-ина

Система чек-ина позволяет игрокам отмечаться каждые N часов в игре или Telegram и получать за это **очки актива**. Очки можно использовать для покупок на сервере через PlaceholderAPI.

#### Как отметиться:

**В игре:**
```
/checkin
```

**В Telegram:**
1. Откройте меню `/menu`
2. Нажмите кнопку "🎯 Отметиться"
3. Или отправьте команду `/checkin`

#### Как работают очки:

- Игрок получает **от 10 до 100 очков** (настраивается) за каждую отметку
- Отмечаться можно раз в **6 часов** (настраивается)
- За ежедневные отметки подряд начисляется **бонус серии** (стрик)
- Очки накапливаются и отображаются через PlaceholderAPI

#### Настройка чек-ина:

```yaml
checkin:
  enabled: true
  cooldown_hours: 6                  # Кулдаун между отметками (часы)
  reward:
    type: "random"                    # random или fixed
    min: 10                           # Минимум очков
    max: 100                          # Максимум очков
    fixed: 50                         # Фиксированное количество (если type: fixed)
  currency_name: "⚡ очков актива"    # Название валюты
  streak:
    enabled: true                     # Включить бонус за серию
    max_bonus: 50                     # Максимальный бонус
  commands:
    in_game: "checkin"                # Команда в игре
    in_game_alias: "ch"               # Алиас команды
    telegram: "checkin"               # Команда в Telegram
  messages:
    success: |
      ✅ **Отметка принята!**
      🎁 Получено: **%points%** %currency%
      🔥 Серия: **%streak%** дней
      💰 Всего: **%total%** %currency%
      ⏰ Следующая отметка через: %cooldown%
    cooldown: |
      ⏰ **Вы уже отмечались!**
      Следующая отметка через: **%time%**
      💰 Ваш баланс: **%points%** %currency%
```

**Плейсхолдеры:**
- `%points%` — получено очков за эту отметку
- `%total%` — всего очков
- `%streak%` — текущая серия дней
- `%cooldown%` — время до следующей отметки
- `%time%` — оставшееся время кулдауна


### Плейсхолдеры PlaceholderAPI

Плагин поддерживает PlaceholderAPI для интеграции с другими плагинами (меню, магазины, квесты).

#### Пример использования:


### Система репутации

Система репутации позволяет игрокам оценивать друг друга положительно или отрицательно. Это помогает создать сообщество и выделить активных/полезных игроков.

#### Как дать репутацию:

**В Telegram:**
1. Ответьте (reply) на сообщение игрока
2. Напишите `/rep+ Причина` для положительной репутации
3. Или `/rep- Причина` для отрицательной репутации
4. Причина обязательна (настраивается в конфиге)

**В игре:**
1. Выполните команду `/rep +PlayerName Причина` для положительной
2. Или `/rep -PlayerName Причина` для отрицательной

#### Ограничения:

- **Кулдаун** - между выдачей репутации одному игроку (по умолчанию 30 минут)
- **Нельзя себе** - нельзя давать репутацию самому себе
- **Причина обязательна** - нужно указать причину (настраивается)
- **Минимальная длина причины** - обычно 5 символов (настраивается)

#### Просмотр репутации:

- `/rep` - ваша репутация
- `/rep PlayerName` - репутация игрока
- `/reptop` - топ игроков по репутации
- `/reprecent` - последние изменения репутации

#### Уровни репутации:

Система автоматически определяет уровень репутации на основе общего рейтинга:
- **Отрицательная** - рейтинг < 0
- **Нейтральная** - рейтинг = 0
- **Положительная** - рейтинг > 0
- **Хорошая** - рейтинг > 10
- **Отличная** - рейтинг > 50
- И т.д. (настраивается)

Система переводов позволяет игрокам переводить деньги друг другу через Telegram. **Требует Vault и плагин экономики!**

### Мини-игра "Угадай слово"

Интерактивная игра, где игрок должен угадать загаданное слово за ограниченное время.

#### Использование:

**В Telegram:**
```
/game
```

**В игровом канале:**
```
/game
```

#### Как играть:

1. Запустите игру командой `/game`
2. Вам покажут загаданное слово с подсказкой (например: `П***о`)
3. У вас есть ограниченное время (по умолчанию 60 секунд)
4. Напишите правильное слово в чат
5. Если угадали - получите награду!
6. Чем быстрее угадаете - тем больше бонус

#### Награды:

- **Базовая награда** - фиксированная сумма (настраивается)
- **Бонус за скорость** - дополнительная награда за быстрое угадывание
- **Максимальный бонус** - ограничение на бонус

#### Настройка:

Настройки находятся в файле `game.yml`:

```yaml
enabled: true
command:
  enabled: true
  auto-delete-seconds: 0           # Автоудаление сообщений (0 = не удалять)

game_settings:
  timeout_seconds: 60               # Время на угадывание
  min_word_length: 4                # Минимальная длина слова
  max_word_length: 10              # Максимальная длина слова
  cooldown_seconds: 30             # Кулдаун между играми
  max_concurrent_games: 10         # Максимум одновременных игр

rewards:
  base_reward: 20                   # Базовая награда
  speed_bonus: 10                   # Бонус за скорость
  max_bonus: 10000                  # Максимальный бонус
  commands:
    - "eco give %player% %reward%"
    - "bc 🎉 %player% выиграл %reward% монет!"
```

### Статистика и топы

Система статистики отслеживает активность игроков и формирует топы.

#### Команды статистики:

- `/stats [период]` - статистика игроков за период
- `/top [период]` - топ по времени игры за период
- `/topbal` - топ по балансу (требует Vault)

#### Периоды:

- `1h`, `5h`, `12h`, `24h` - часы
- `1d`, `7d`, `30d` - дни
- `1w`, `2w`, `4w` - недели
- `1m`, `3m`, `6m` - месяцы

#### Автоматические уведомления:

Плагин может автоматически отправлять топы в указанное время:

```yaml
auto_notifications:
  enabled: true
  playtime_top:
    enabled: true
    period: "1d"                    # Период для топа
    schedule: "12:00,20:00"         # Время отправки (можно несколько)
    title: "🏆 **Ежедневный топ активности**"
    footer: "\n⏰ Обновляется автоматически"
    
  balance_top:
    enabled: true
    schedule: "18:00"                # Время отправки
    title: "💰 **Ежедневный топ богачей**"
    footer: "\n💸 Стань богаче и попади в топ!"
```

#### Исключение игроков из топов:

Можно исключить администрацию из топов:

```yaml
auto_notifications:
  playtime_top:
    exclude:
      enabled: true
      permissions:
        - "group.admin"
        - "group.moderator"
        - "ztelegram.top.exclude"
```

#### Награды за топ-1:

Можно настроить награды для игроков, занявших первое место:

```yaml
auto_notifications:
  playtime_top:
    rewards:
      enabled: true
      list:
        - name: "5 алмазов"
          commands:
            - "give %player% diamond 5"
            - "broadcast &6&l🏆 &e%player% &6получил 5 алмазов!"
```

## ❓ Частые вопросы (FAQ)

### ❓ Плагин не работает, что делать?

1. **Проверьте Vault** - убедитесь, что Vault установлен и загружен
2. **Проверьте токен бота** - токен должен быть правильным
3. **Проверьте ID каналов** - ID должны быть правильными
4. **Проверьте права бота** - бот должен быть администратором в каналах
5. **Проверьте консоль** - там могут быть ошибки

### ❓ Как изменить формат сообщений?

Отредактируйте соответствующие параметры в `config.yml`:
- Для сообщений из Telegram в игру: `chat.telegram-to-minecraft.format`
- Для сообщений из игры в Telegram: `chat.minecraft-to-telegram.format`

### ❓ Команда `/telegram reload` вызывает ошибки

Это ожидаемое поведение. Рекомендуется использовать стандартную перезагрузку сервера вместо `/telegram reload`.

### ❓ Как отключить уведомления о входе/выходе игроков?

Установите параметры `enabled: false` в секции `chat.player-events`:

```yaml
chat:
  player-events:
    join:
      enabled: false
    quit:
      enabled: false
```

### ❓ Как настроить время автоудаления сообщений?

Измените значение параметра `auto-delete-seconds` в соответствующих секциях:

```yaml
commands:
  auto-delete-seconds: 30  # 0 = не удалять
```

### ❓ Топ по балансу не работает

**Требуется Vault!** Убедитесь, что:
1. Vault установлен и загружен
2. Установлен плагин экономики (EssentialsX, CMI, или другой)
3. Vault подключен к плагину экономики

### ❓ Как получить свой Telegram ID?

1. Напишите боту @userinfobot в Telegram
2. Отправьте `/start`
3. Бот покажет ваш ID

### ❓ Как использовать темы (topics) в каналах?

Добавьте `_номер_темы` в конец ID канала:
- Обычный канал: `-1001234567890`
- Тема в канале: `-1001234567890_123456`

### ❓ Игроки не могут зарегистрироваться

Проверьте:
1. Игрок должен быть **онлайн** на сервере
2. Никнейм должен быть правильным (без пробелов, только английские буквы, цифры и `_`)
3. Игрок не должен быть уже зарегистрирован
4. Канал регистрации должен быть правильно настроен
